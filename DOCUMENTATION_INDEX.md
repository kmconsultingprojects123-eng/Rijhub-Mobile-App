## 📝 Notes

### Naming Convention
- `_artisanServices` - Local state variable
- `artisan['_artisanServices']` - Cached in artisan object
- `subCategoryName` - Always use this field for display

### Key Endpoints
- `GET /api/artisan-services/me` - Fetch artisan services
- `GET /api/artisans/{id}` - Fetch artisan details
- `POST /api/artisan-services` - Create/update services

### Important Classes
- `MyServiceService` - Service fetching
- `ArtisanDetailPageWidget` - Service loading
- `HomePageWidget` - Service display in cards
- `DiscoverPageWidget` - Service extraction
- `SearchPageWidget` - Service display
